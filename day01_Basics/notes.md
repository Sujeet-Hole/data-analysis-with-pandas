# Day 01 — Pandas Basics

## Topics Learned

* Read CSV files
* Use head() and tail()
* Check dataset size with shape
* Find null values
* Replace null values
* Filter rows using conditions
* Use iloc for row access
* Use groupby()
* Use value_counts()
* Find mean, max, min, median, and sum
* Sort data

## Datasets Used

* Student Dataset (h.csv)
* Online Retail Dataset

## Key Learning

* Pandas is used to work with table data.
* Data is stored in rows and columns.
* Null values can be found and replaced.
* Filtering helps get required rows.
* groupby() helps analyze data by category.
* iloc() accesses rows by position.
* Basic statistics help understand data.

## Note

Changes made in Pandas do not change the original CSV file unless the file is saved again.
